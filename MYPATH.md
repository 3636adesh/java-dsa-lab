# DSA Learning Plan (Java) 🚀

## Goal
Build strong Data Structures and Algorithms (DSA) concepts using **Java**, with daily problem solving and structured learning.

---

## Learning Strategy

1. Learn concept from YouTube
2. Solve questions from LeetCode
3. Use Striver A2Z Sheet for structured practice
4. Ask ChatGPT for similar questions on the same concept
5. Practice difficult problems weekly

---

## Daily Plan 📅

### Daily Questions
- 1 Easy Question
- 4 Medium Questions
- ChatGPT: Ask for similar concept questions

### Weekly Questions
- Minimum 2 Hard Questions

---

## Daily Workflow

### Step 1: Learn Concept
- Watch YouTube video
- Read Striver A2Z topic
- Read Java Bootcamp notes

### Step 2: Practice
- Solve 1 Easy Question
- Solve 4 Medium Questions
- Ask ChatGPT:
  > Give me similar questions on this concept

### Step 3: Weekly Challenge
- Solve 2 Hard Questions from:
  - Striver Sheet
  - 450 DSA Questions
  - LeetCode Hard

---

## Weekly Structure

### Monday to Friday
- Concept Learning
- Easy + Medium Questions

### Saturday
- Revision
- Medium Questions
- 1 Hard Question

### Sunday
- Revision
- 1 Hard Question
- Analyze mistakes
- Improve weak topics

## ChatGPT Usage 🤖

Use ChatGPT like this:

**Example Prompt:**

```
I learned Binary Search from Striver A2Z sheet.
Give me 5 LeetCode questions on Binary Search:
1 Easy
4 Medium
```

**For Hard Questions:**

```
Give me 2 Hard Binary Search problems from LeetCode
```
